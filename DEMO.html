🏦 LOCDS

Loan Origination and Credit Decisioning System

Application Built & Ready for Deployment

📋 Application Overview

LOCDS is an enterprise-grade ASP.NET WebForms application for managing loan applications, credit scoring, underwriting decisions, and loan offers. The system implements industry best practices for security, error handling, logging, and maintainability.

✓ Status: Application built with security hardening, error handling, IoC container, NUnit tests, and deployment artifacts.

🏗️ System Architecture

Layered Structure

  • LOCDS.Web - ASP.NET WebForms UI + Security + Global Error Handling
  • LOCDS.BLL - Business Services (Scoring, Underwriting, Offers, Auditing)
  • LOCDS.DAL - Data Access & Repository Pattern
  • LOCDS.Common - Shared Constants, Enums, Exceptions, Infrastructure
  • LOCDS.Entities - Domain Models
  • LOCDS.Tests - NUnit + Moq Test Suite

Technology Stack

  • .NET Framework: 4.8
  • Web Platform: ASP.NET WebForms
  • Database: SQL Server
  • DI Container: Unity IoC
  • Logging: log4net
  • Testing: NUnit + Moq
  • Validation: FluentValidation

🔒 Security Features Implemented

✓ Role-Based Access Control

Custom HTTP module enforcing role-based authorization at request level before page execution.

✓ Anti-CSRF Protection

Cookie-backed token validation on all POST requests with fixed-time comparison.

✓ SQL Injection Prevention

Parameterized queries throughout DAL layer. AntiXss encoder enabled in Web.config.

✓ Session Fixation Prevention

Session ID regeneration after successful authentication via SessionIDManager.

✓ Data Masking

PAN and Aadhaar masking in UI output to protect sensitive applicant data.

✓ HTTPS + Security Headers

Automatic HTTPS redirect, X-Frame-Options, CSP, HSTS, X-Content-Type-Options headers.

🛠️ Enterprise Error Handling

  • Global Exception Capture: Application_Error in Global.asax catches unhandled exceptions
  • Correlation ID: AsyncLocal-backed context for request tracing across layers
  • Branded Error Pages: Custom 403, 404, 500 pages with bank branding
  • UpdatePanel Error Handling: Async postback errors logged and user-friendly messages returned
  • Exception Hierarchy: Business exceptions (AppException, LoanException, BureauException, UnderwritingException)
  • Structured Logging: log4net with correlation ID, user, session, URL, and timestamp in all entries

🚀 To Run This Application Locally

Prerequisites

  • Windows with .NET Framework 4.8
  • SQL Server (Express or higher)
  • IIS Express or IIS with ASP.NET 4.8 enabled
  • Visual Studio 2022 (or Build Tools + MSBuild)

Setup Steps

  1. Set connection string: 
    $env:LOCDS_CONNECTION_STRING = "Server=localhost;Database=LOCDS;Trusted_Connection=True;Encrypt=False;"
  2. Build the solution: 
    msbuild LOCDS.sln /p:Configuration=Debug /p:Platform="Any CPU" /v:minimal
  3. Run database migrations: Execute scripts in LOCDS/LOCDS.DAL/Database/ in order
  4. Run tests: 
    dotnet test LOCDS/LOCDS.Tests/LOCDS.Tests.csproj
  5. Start IIS Express from Visual Studio: Open LOCDS.sln in VS → Press F5 or use "Run"

For Production Deployment

./scripts/deploy.ps1 -IisSiteName "Default Web Site/LOCDS" -Configuration Release
